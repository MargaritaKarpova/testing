Кейсы проверок мобильных приложений

1. Проверка установки приложения с нуля.
2. Проверка запуска приложения.
3. Проверка удаления приложения.
4. Проверка повторной установки приложения (после удаления).
5. Проверка обновления приложения на новую версию (запустить, проверить сохраненные встроенные покупки, авторизацию, дизайн, файлы, прогрессы, контакты и др.).
6. Проверка платной версии.
7. Проверка окончания периода оплаты (переход на бесплатный контент).
8. Проверка главного экрана приложения при открытии.
9. Проверка работы приложения во время подключения зарядки к устройству.
10. Проверка работы приложения во время отключения зарядки к устройству.
11. Проверка работы приложения с наушниками и во время подключения/отключения наушников к устройству.
12. Проверка работы приложения во время уровня заряда менее 50%.
13. Проверка работы приложения во время уровня заряда менее 20%.
14. Проверка работы приложения во время уровня заряда менее 10%.
15. Проверка работы приложения при прерываниях (входящий звонок/смс/push/будильник/режим «Не беспокоить» и т.д.).
16. Проверка работы приложения при прерываниях
17. Проверка работы приложения во время телефонного разговора.
18. Проверка работы приложения при сворачивании приложения.
19 Проверка работы приложения при разворачивании приложения.
20. Проверка работы приложения в фоновом режиме.
21. Проверка закрытия приложения (кильнуть приложение).
22. Проверка закрытия приложения из свернутого состояния (свернуть и кильнуть).
23. Проверка работы приложения в портретной ориентации.
24. Проверка работы приложения в альбомной ориентации.
25. Проверка работы приложения при смене ориентации экрана телефона.
26. Проверка закрытия приложения из разных ориентациях экрана телефона.
27. Проверка работы приложения по Wi-Fi.
28. Проверка работы приложения по мобильному интернету.
29. Проверка работы приложения при переподключении с мобильного интернета на  Wi-Fi.
30. Проверка работы приложения при переподключении с Wi-Fi на  мобильный интернет.
31. Проверка работы приложения при прерывании интернета.
32. Проверка работы приложения при переключении с E на 3G, с 3G на 4G, с 4G на 5G и обратно.
33. Проверка работы приложения при извлечении SIM-карты.
34. Проверка работы приложения при извлечении карты памяти.
35. Проверка работы приложения при подключении SIM-карты.
36. Проверка работы приложения при подключении карты памяти.
37. Проверка работы приложения в различных разрешениях экрана.
38. Проверка работы приложения при включении/отключении GPS.
39. Проверка работы приложения при темной/светлой и других тем телефона.
40. Проверка работы приложения большом количестве взаимодействий пользователя с приложением.
